# Product Documentation (Overview)

## Tiers
- **Standard**: core platform, reporting, email support.
- **Enterprise**: everything in Standard plus SSO/SAML, advanced security,
  role-based access control, priority support, and a dedicated CSM.

## Security
Enterprise includes SSO, audit logs, and SCIM provisioning - designed to meet
enterprise security review requirements.

## Adoption
Admin training workshops measurably increase feature adoption and correlate
with higher renewal rates.
